# 1929. Concatenation of Array

**Difficulty:** Easy

**Topics:** Arrays

**Link:** https://leetcode.com/problems/concatenation-of-array/

## Approach
The project simply instructs you to concatenate two int arrays. This could be accomplished in many ways in python, but I opted for the simple 1 line solution. However, I could had used the addition operand, 'list.extend(list)' or a for
loop.

## Complexity
- Time: O(n)
- Space: O(n)
